# 🧮 Calculator Web App

## 📖 Description
This is a responsive and interactive **Calculator Web Application** built using HTML, CSS, and JavaScript.  
It performs basic arithmetic operations with a clean user interface and smooth user experience.

The calculator is designed to mimic real-world calculator functionality with intuitive button layout and responsive design.

---

## 🚀 Features

- ➕ Basic operations: Addition, Subtraction, Multiplication, Division  
- 🧹 Clear display (C button)  
- ⬅️ Delete last character  
- 📊 Percentage calculation (%)  
- 🔢 Decimal number support  
- ⚡ Instant result calculation  
- 📱 Fully responsive design  
- 🎨 Modern UI with icons (Font Awesome)  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript  
- Font Awesome (icons)
